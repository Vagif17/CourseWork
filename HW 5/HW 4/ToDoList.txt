Сущности:
	Customer:
		Id: 		int
		Name:		string
		Address:	string?
		Email:		string		 
		PhoneNumber: 	string?
		CreatedAt:	DateTimeOffset
		UpdatetAt: 	DateTimeOffset // Обновляеться при любом изменении сущности
		DeletedAt:	DateTimeOffset? // Используеться для soft delete
		
	Invoice:
		Id:		int
		CustomerId:	int
		StarDate: 	DateTimeOffset // Начало даты выполнения работ
		EndDate:	DateTimeOffset // Конец периода выполнения работ
		Rows:		InvoiceRow []
		TotalSum:	Decimal // сумма Sum всех Rows
		Comment:	string?
		Status:		InnvoiceStatus(Created, Sent, Received, Paid, Cancelled, Rejected) 
		CreatedAt:	DateTimeOffset
		UpdatetAt: 	DateTimeOffset // Обновляеться при любом изменении сущности
		DeletedAt:	DateTimeOffset? // Используеться для soft delete

		
	InvoiceRow :
		Id: 		int
		InnvoiceId:	int
		Service:	string // Название выполненной работы
		Quantity:	decimal // Количество единиц выполненной работы
		Rate:		decimal // стоимость одной единицы выполненной работы
		Sum:		decimal // общая стоимость выполненной работы на основании Rate и Quantity





API должен поддерживать следующие операции:
	Клиенты:
		1. Добавление клиента +
		2. Редактирование клиента +
		3. Удаление клиента (hard delete) + // Удалять можно только тех клиентов, которым не было отправлено ни одного инвойса
		4. Архивирование клиента + (soft delete)
		5. Получение списка клиентов +
		6. Получения клиента по Id + 


	Инвойсы:
		1. Создание инвойса +
		2. Редактирование инвойса + - Редактировать можно только те инвойсы которые пока не отправлены 
		3. Изменение статуса инвойса +
		4. Удаление инвойса (hard delete) + - удалять можно только те инвойсы которые пока не отправлены
		5. Архивирование инвойса + (soft delete) 
		6. Получение списка инвойсов +
		7. Получения инвойса по Id +
