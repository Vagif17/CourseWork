# TextMeMessenger

![Logo](https://res.cloudinary.com/diq4utz5c/image/upload/v1772545700/TextMeMessenger%28MainLogo%29.png)

**TextMe** — это приложение-альтернатива таким приложениям, как WhatsApp, Telegram, Max и Messenger.
**TextMe** объединяет в себе лучшие качества нескольких мессенджеров в одном приложении. Переписывайтесь, отправляйте изображения, делитесь файлами, созванивайтесь, читайте каналы и собирайтесь с друзьями в групповых чатах и звонках.
